HTML基础笔记

一、排版标签

  1. 标签由<、>、\、英文单词或字母组成，并且把标签中<>包括起来的英文单词或字母称为标签名
  2. 常见标签由两部分组成，我们称之为:双标签，前部分叫开始标签，后部分叫结束标签，两部分之间包裹内容
  3. 少数标签由一部分组成，我们称之为:单标签自成一体,无法包裹内容
  4. （标签）标题标签<h1>...</h1><h2>...<\h2>........<h6>...<\h6> 共六级 由大到小
  5. （标签）<p>....</p> 段落标签 双标签 有头有尾 放入 body 里
  6. （标签）<br>换行标签 单标签 文字强制换行
  7. （标签）<hr>水平分割线 可用作不同主体之间的分隔 加入一条线

二、文本格式化标签

  1. 加粗、下划线、删除线、倾斜
  2. （标签）<b>：加粗 <u>：下划线 <i>：倾斜 <s>：删除线
  3. （标签）<strong>：加粗 <ins>：下划线 <em>：倾斜 <del>：删除线突出重要性语境使用

三、媒体标签

  1. （标签，属性）<img src="" alt="" title="">** 加入图片**
    • 当网页和图片在同一目录下时，路径直接写图片名字即可
    • alt 为替换文本，当图片加载失败时以 alt 的文本代替
    • title：当鼠标悬停在图片上方时出现的文本
  2. （属性）width/height**：设置图片的宽度和高度 **
    • 只设置一个，另一个会自动等比例缩放
    • 都设置，若设置不当图片会变形
  3. 属性注意点：
    • 一个标签可以存在多个属性
    • 属性之间没有固定顺序
    • 属性写在开始标签内部
    • 标签名和属性之间必须以空格分开
    • 一个标签可以存在多个属性
  4. 绝对路径（无法多端使用）
  5. **相对路径：从当前文件开始查找目标文件 **
    • 同级目录，”./“为当前（同级）
    • 上级目录，../（到上级目录）
    • 下级目录，同级文件夹名.文件名
  6. （标签）<audio src=""></audio>：音频标签
    1. 属性：
      • sec 地址
      • controls 显示播放控件
      • autoplay 自动播放
      • loop 循环播放
  7. （标签，属性）<video src=""></video>：视频标签
    1. 属性：
      • sec 地址
      • controls 显示播放控件
      • autoplay 自动播放(谷歌需配合 muted 实现静音播放)
      • oop 循环播放

四、链接标签

  1. （标签，属性）<a href="./目标网页.html">内容</a>：超链接、锚标签、a标签
    • 地址为 **# **是空连接
    • 属性：
      • target：控制目标网页的打开方式
        • “_self”：默认值，覆盖原网页
        • “_blank”：在新窗口跳转

五、列表

  1. 场景：排序整齐，如新闻列表、排行榜、账单等**
    • 特点：按照行的方式，整齐显示内容
  2. 内容：
    • 无序列表、有序列表、自定义列表
  3. （标签）无序列表：ul标签
    • 注意点：
      • ul 标签内只允许有 li 标签
      • li 标签可以包含任何内容
      • 列表默认开头为圆点·
  4. （标签）有序列表：ol标签
    • 注意点：
      • ol 标签内只允许有 li 标签
      • li 标签可以包含任何内容
      • 列表默认开头为序号标识
  5. （标签）自定义列表：dl 标签 -注意点：dl 标签内只允许有 dt、dd 标签 - dt、dd 标签可以包含任何内容 - dd 标签前默认显示缩进效果 - dt 标签表示自定义列表的主题 - dd 标签表示自定义列表的针对主题的每一项内容

六、表格

  1. 以行和列形式，整齐地展示数据
  2. （标签）基本标签 ：
    • table（表格整体，包括多个 tr）
    • tr(表格每行，可用于包裹 td)
    • td（表格单元格，可用于包裹内容）
    • table > tr > td
  3. （属性）表格相关属性：
    • border：边框宽度
    • width ：表格宽度
    • height：表格高度
    • 实际开发时推荐使用 CSS 设置效果
  4. （标签）表格标题和表头单元格标签：
    • 整体大标题、一列小标题
      • caption：表格大标题
        • 默认为表格整体顶部居中位置显示
      • th：表头单元格
        • 表示一列小标题，通常用于表格第一行，默认内部文字加粗并居中显示
      • 注意点：caption 标签书写在 table 标签内部
      • th 标签书写在 tr 标签内部
        • 用于替换 td 标签
  5. （标签）表格的结构标签：
    • thead：表格头部
    • tbody：表格主题
    • tfoot：表格底部
    • 注意点：表格的结构标签内部可包含 tr 标签，表格的结构标签可以省略
  6. （属性）合并单元格：
  • 步骤：
      1. 明确合并哪几个单元格
      1. 通过左上原则，确定保留谁、删除谁
      • 上下合并：只保留最上的，删除其他
      • 左右合并：只保留最左的，删除其他
      1. 跨行合并、跨列合并
      • rowspan + 合并单元格的个数 ：跨行合并（上下合并）
      • colspan + 合并单元格的个数： 跨列合并（左右合并）
      1. 注意点：只能在同一个结构标签内的单元格才能合并，不能跨标签合并（thead/tbody/tfoot）

7 . （应用）优秀学生信息表格

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    <table border="3" width="300" height="400">
        <caption><b>优秀学生信息表格</caption>
            <thead>
                <tr>
                    <th>年级</th>
                    <th>姓名</th>
                    <th>学号</th>
                    <th>班级</th>
                </tr>
            </thead>
            <tbody>
                <tr>
                    <td rowspan="2">高三</td>
                    <td>张三</td>
                    <td>110</td>
                    <td>三年二班</td>
                </tr>
                <tr>
                    <td>赵四</td>
                    <td>120</td>
                    <td>三年二班</td>
                </tr>
            </tbody>
            <tfoot>
                <tr>
                    <td>评语</td>
                    <td colspan="3">你们都很优秀</td>
                </tr>
            </tfoot>
    </table>
</body>
</html>

七、表单

  1. 例如：搜索框、填写信息框
  2. （标签）imput 系列标签：
    • （属性）type 属性： - （属性值）①：text -> 文本框，用于输入单行文本 - （属性值）②：password -> 密码框，用于输入密码 - （属性值）③：radio -> 单选框，用于多选一 - （属性值）④：checkbox-> 多选框，用于多选多 - （属性值）⑤：file -> 文件选择，用于之后上传文件 - （属性值）⑥：submit -> 提交摁钮，用于提交 - （属性值）⑦：reset-> 重置摁钮，用于重置 - （属性值）⑧：button -> 普通摁钮，默认无功能，之后配合 JS 添加功能
    • （属性）placeholder`属性：【配合 txt、password 使用】
      • 占位符 显示提示字符
    • （属性）name 属性：【配合 radio 使用】
      • 分组，有相同 name 值的单选框为一组，一组中同时只能有一个被选中
    • （属性）checked属性：【配合 radio、checkbox 使用】
      • 默认选中，没有属性值
    • （属性）mutiple属性：【配合 file 使用】
      • 多文件选择
    • （属性）value属性：【配合 submit、reset、button 使用】
      • 更改摁钮中的内容
    • （标签）form预标签：
      • 实现提交，重置，自定义摁钮的功能
  3. （标签）button 标签：
  • 谷歌浏览器中 button 默认是提交摁钮，button 标签是双标签，更便于包裹其他内容：文字、图片等
    • （属性）submit属性：提交至服务器
    • （属性）reset属性：重置信息
    • （属性）button属性：普通摁钮
  1. （标签）select 标签：
    • 下拉菜单的整体
    • （标签）option标签：
      • 下拉菜单的每一项
    • （属性）**selected**属性：
      • 下拉菜单默认选中
  2. （标签）textarea 标签：
  • 文本域标签，可输入多行文本的控件
    • （属性）cols属性：
      • 规定文本域的可见宽度
    • （属性）rows属性：
      • 规定文本域的可见行数
    • 注意点：右下角可拖拽改变大小，实际开发时针对样式效果推荐使用#CSS
  1. （标签）label 标签：
  • 使用方法 ①： 1. <input type="radio" id="xxx"><label for="xxx">选项名称</label> 1. 使用 label 标签把内容(如文本)包裹起来 1. 在表单标签上添加 id 属性 1. 在 label 标签的 for 属性中设置对应的 id 属性值
  • 使用方法 ②: 1. <label><input type="radio">选项名称</label> 1. 直接使用 label 标签把内容(如文本)和表单标签一起包裹起来 1. 需要把 label 标签的 for 属 性删除即可

八、语义化标签

  1. 没有语义的布局标签
    • 场景：实际开发网页会大量使用 div 和 span 两个没有语义的标签
    • （标签）div标签：一行只显示一个 - 独占一行 - 自动换行
    • （标签）span标签：一行可以显示多个
  2. 有语义的布局标签
    • 手机端网页制作时使用
    • 在 HTML5 新版本，推出一些有语义的标签使用
    • （标签）header标签： 网页头部
    • （标签）nav标签： 网页导航
    • （标签）footer标签： 网页底部
    • （标签）aside标签： 网页侧边栏
    • （标签）section标签：网页区块
    • （标签）article标签：网页文章
    • 注意点：以上显示与 div 标签一致，但是比 div 标签多了不同的语义
