✍️Flex 布局/弹性布局

  • 是一种浏览器提倡的布局模型
  • 布局网页更简单、灵活
  • 避免浮动脱标的问题
  • 作用：
    • 灵活控制块级盒子的布局方式，避免浮动布局中脱离文档流现象发生
    • Flex 布局非常适合结构化布局
  • 设置方式：
    • 父元素添加display: flex;，子元素可以自动的挤压或拉伸
  • 组成部分：
    • 弹性容器（添加 flex 的父元素）
    • 弹性盒子
    • 主轴
    • 侧轴 / 交叉轴

✍️ 修改主轴对齐方式

  • 属性名：justify-content
  • 属性值：

✍️ 修改侧轴对齐方式

  • 属性名：align-items（添加在弹性容器）
  • 属性名：align-self控制某个弹性盒子在侧轴的对齐方式（添加在弹性盒子）
  • 属性值：

✍️ 弹性伸缩比

  • 属性名：flex: 值;
  • 取值分类：
    • 数值（整数）
  • 注意：
    • 只占用父盒子剩余尺寸（把未设置 flex 的去掉）

✍️ 修改主轴方向

  • 主轴默认是水平方向，侧轴默认是垂直方向

  • 更改之后，侧轴方向也变化

  • 属性名：flex: direction

  • 属性值：

  • 默认情况下不换行，也就是就算宽度超出，也会自动缩小（弹性)

  • 换行显示：flex-wrap: wrap;

    • 属性值默认是 nowrap，即不换行

  • 行对齐方式：align-content: ..;

    • 取值和justify-content基本相同 （没有空白区域全部相同的evenly）

CSS 3

✍️1 字体图标

  • 目标：使用字体图标技巧实现网页中简洁的图标效果
  • 直接将字体图标文件夹放在根目录即可
  • 步骤：
    • 引样式表
      • 在 head 标签内用 link 标签引用
    • 调用类名
      • class="iconfont 字体图标类名"iconfont 是固定的，不能改变

  • 上传矢量图：图库中没有项目所需的图标
    • iconfont 网站上传矢量图生成字体图标
    • 将 SVG 格式图标格式上传

✍️2 过渡

  • 作用：让元素样式慢慢的变化，常配合 hover 使用，增强网页交互体验
  • 属性名：translation
  • 常见取值：
    • 使用具体属性过渡时，格式：属性 1 时长，属性 2 时长，属性 3 时长…..
  • 注意点：
    • 过度需要：默认状态 和 hover 状态样式不同，才有过渡效果
    • translation 属性经常给需要过渡的元素本身加（移入移除都有效果），并非加在 hover 伪类中
    • translation 属性设置在不同元素中，效果不同：
      • 给默认状态设置，鼠标移入移除都有过渡效果
      • 给 hover 状态设置，鼠标移入有过渡效果，移出没有过渡效果

✍️3 平面转换

  • 使用transform属性实现元素的位移、缩放、旋转等效果
  • 改变盒子在平面的形态
  • 2D 转换

✍️4 平面转换-位移

  • 语法：transform:translate(水平移动距离，垂直移动距离);
  • 取值：取值正负均可
    • 像素单位数值
    • 百分比（参照物为盒子自身尺寸）
    • 注意：x 轴正向为右，y 轴正向为下
  • 技巧：
    • translate 只给一个数值，表示 x 轴方向移动距离
    • 单独设置某个方向的移动距离：translateX() & translate()

✍️5 平面转换-旋转

  • 默认为围绕盒子中心旋转：
    • 语法：transform:rotate(角度);
      • 角度单位是 deg
    • 技巧：取值正负均可
      • 正值表示顺时针
      • 负值表示逆时针
      • 搭配过渡transition才有效果
  • 改变旋转的原点：
    • 语法：transform-origin:原点水平位置 原点垂直位置;
    • 不是放在 hover 伪类里，代替 transition
    • 取值：
      • 方位名词：left、top、right、bottom、center
      • 像素单位数值
      • 百分比（参照盒子自身尺寸计算）

✍️6 平面转换-缩放

  • 使用 scale 改变元素的尺寸
  • 实现从中间缩放，若使用宽高来调节，不能从中间缩放
  • 语法：transform:scale(X轴缩放倍数，Y轴缩放倍数);
  • 技巧：
    • 一般情况下，只为 scale 设置一个值，表示 x 轴和 y 轴等比例缩放
      • transform:scale(缩放倍数);

✍️7 平面转换-渐变

  • 使用 background-image 属性实现渐变背景效果

  • 转换中的渐变：

    • background-image: linear-gradient(颜色1，颜色2，.....);

      • 这个是从透明到黑色的渐变

✍️8 多重转换

  • 复合属性实现多形态转换
  • 语法：transform:translate() ratate() scale();
  • 把旋转放在后边，如果放在前面，位移方向会受到影响（改变了坐标轴向）
  • 注意：其实每个渐变属性都是一种装饰，并不是专门为转换服务

💪9 空间转换

  • 使用transform属性实现元素在空间的位移、缩放、旋转等效果
  • 改变盒子在空间的形态，X、Y、Z 轴，Z 轴指向自己
  • 3D 转换

💪10 空间转换-位移

  • 语法transform: translate3d(x,y,z);
    • transform: translateX();
    • transform: translateY();
    • transform: translateZ();
  • 取值：正负均可
    • 像素单位数值
    • 百分比

💪11 空间转换-透视

  • 属性：（添加给父级）
    • perspective: 值;
    • 取值：像素单位数值，数值一般在 800 - 1200

💪12 空间转换-旋转

  • transform: rotateZ(值);绕 Z 轴旋转
  • transform: rotateX(值);绕 X 轴旋转
  • transform: rotateY(值);绕 Y 轴旋转
  • 自定义旋转轴：
    • rotate3d(x,y,z,角度度数)：用来设置自定义旋转轴的位置及旋转角度
    • X、Y、Z 取值为 0 - 1 之间的数字

💪13 空间转换-左手法则

  • 判断旋转方向：
    • 左手握住旋转轴，拇指指向正值方向，手指弯曲方向为旋转正值方向

💪14 空间转换-立体呈现

  • 语法：（添加给父级）
    • transform-style：preserve-3d;
    • 使子元素处于真正 3D 空间
    • 默认值 flat，表示子元素处于 2D 平面内呈现

💪15 空间转换-缩放

  • 语法：
    • transform：scaleX(倍数);
    • transform：scaleY(倍数);
    • transform：scaleZ(倍数);
    • transform：scale3d(x, y, z);

🌾17 动画

  • 过渡只能实现两个状态之间的变化
  • 动画：实现多个效果之间的变化过程，动画过程可控（重复播放，最终动画，是否暂停）
  • 使用 animation 添加动画效果
    • 构成动画的最小单位： 帧或动画帧
  • 定义步骤：
    • 动画实现步骤：
      • 定义动画：

     - 使用动画：`animation：动画名称 动画花费时长;`
        - 动画使用既可以复合也可以多组，使用逗号隔开。
        - `animation：动画名称 动画花费时长，动画名称 动画花费时长...... ;`

🌾18 from-to

  • from{CSS代码}
  • to{CSS代码}
    • 如果动画开始状态和盒子默认状态相同，可以省略开头的from{CSS代码}

🌾19 百分比

  • 百分比指的是动画总时长的占比
  • 从一个到另一个的过渡时长

🌾20 animation 复合属性

  • animation：**动画名称 动画时长** 速度曲线 延迟时间 重复次数 动画方向 执行完毕状态;
  • 注意：
    • 动画名称 和 动画时长 必须赋值
    • 取值不分先后顺序
    • 如果有两个时间值，第一个时间表示动画时长，第二个时间表示延迟时间
  • 例子：
    • 速度曲线：
      • linear表示匀速
      • step(数值)表示将时间分为几部分，分步播放
    • 重复次数：
      • 写多大就重复几次
      • 写infinite表示无限循环
    • 动画方向：
      • alternate表示轮流播放，也就是方向变的交替播放
    • 执行完毕状态：
      • 必须把 重复次数 和 动画方向 去掉
      • backwards是默认值，也就是返回初始状态
      • forwards表示停留在最终的状态
